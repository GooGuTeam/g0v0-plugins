# bancho.py API

This plugin provides APIs (v1) from [bancho.py API](https://github.com/JKBGL/gulag-api-docs).

All APIs are prefixed with `/api/plugins/banchopy_api/`. Just replace 

```text
https://api.yourdomain.com/
```

with

```text
https://yourdomain.com/api/plugins/banchopy_api/
```

v2 APIs works in progress, and will be added in the future.
